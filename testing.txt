Category Model
- fields : category name[required]
- if category name exists then,status:"This category already exists".

Brand Model
- fields : category[required],brand_name[required]
- if you try to add same brand under a category more then once, status:"this brand name already exists for the category"

Model Model
- fields : category[required],brand[required],model_name[required]
- if model name exists then,status:"this model already exists".

VehicleType Model
- fields : vehicle type[required]
- if vehicletype exists then,status:"This vehicletype already exists".

User Model
- fields : username[required],phone_no[required],address,email[required],type,status
- phone_no should have exactly 10 digit
- email must have .com extension.
- phone_no must be unique else status:"this phone_no already exists".
- email must be unique else status:"this email already exists".

RideType Model
- fields : name[required]
- if RideType name exists then,status:"This RideType already exists".

Customer Model
- fields : customer_name[required],phone_number[required],address,email[required]
- phone_no should have exactly 10 digit
- email must have .com extension.
- phone_no must be unique else status:"this phone_no already exists".
- email must be unique else status:"this email already exists".

Driver Model
- fields : driver_name[required],phone_number[required],address,email[required],document_verification_type[required],document_link[required]
- phone_no should have exactly 10 digit
- email must have .com extension.
- phone_no must be unique else status:"this phone_no already exists".
- email must be unique else status:"this email already exists".

VehicleOwner Model
- fields : name[required],phone_number[required],address,email[required],document_verification_type[required],document_link[required]
- phone_no should have exactly 10 digit
- email must have .com extension.
- phone_no must be unique else status:"this phone_no already exists".
- email must be unique else status:"this email already exists".

Vehicle Model
- fields : Vehicle_Number[required],model[required],color[required],vehicle_owner[required],vehicle_type[required]
- Vehicle_Number must be unique else status:"this Vehicle_Number already exists".
- Vehicle format

RideDetails Model
- fields : customer[required],vehicle[required],ridetype[required],source[required],destination[required],total_fare[required],gst,driver charge,petrol charge[required],ride status[required]
